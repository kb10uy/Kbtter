動作環境
--------------------------------------------------------------------------------
Windows 7以降
.NET Framework4.5以降
1GB以上のRAM
--------------------------------------------------------------------------------

概要
--------------------------------------------------------------------------------
Kbtterは、WPFをふんだんに使った新しい(?)Twitterクライアントです。
それなりの機能は取り揃えています。
また、IronPythonを使用したプラグインを作成し、運用することが可能です。
--------------------------------------------------------------------------------

使い方
--------------------------------------------------------------------------------
初回起動時、ユーザーの認証を行います。ブラウザで認証のページが開くので、
認証して、PINコードを入力して、OKを押してください。
この手順は、2回目以降の起動は不要です。また、user.acを削除すると、再認証
することが出来ます。

あとはほぼ一般的なクライアントと同じような操作が出来ます。
・右側のユーザー表示欄の各数字をクリックすると、対応したツイート・ユーザーを
  表示します。
・タイムライン・フォロ(ワ)ーの画面で、ユーザーアイコンをクリックすると
  そのユーザーの情報を表示できます。
・上部の簡易ツイート欄では、1行・返信不可という制限付きですが、
  Enterですぐにツイートすることが出来ます。
